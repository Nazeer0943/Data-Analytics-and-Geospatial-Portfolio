# 🌍 Health Infrastructure Distribution & Spatial Analysis

## 📌 Project Overview
This project focuses on identifying community healthcare coverage gaps and optimizing resource allocation through high-precision geospatial visualization and vector digitization. By mapping existing health facilities against regional populations, this spatial analysis acts as a blueprint for targeted intervention planning.

> 🏆 **Achievement:** This analysis was awarded **3rd Place in the DSN World GIS Map Challenge** for outstanding cartographic design, regional print layout, and resource optimization insights.

---

## 🛠️ Geospatial Toolset & Methodologies
*   **Software:** QGIS / ArcMap
*   **Vector Digitization:** Tracing and creating high-accuracy Point (facilities), Line (transportation networks), and Polygon (administrative boundaries) layers.
*   **Spatial Analysis:** Spatial joins, distance calculations, and proximity buffer zone mapping to isolate underserved populations.
*   **Cartography:** Managing Coordinate Reference Systems (CRS) and structuring production-ready map layouts with interactive grids, scales, and legends.

## 📈 Key Insights & Deliverables
1. **Coverage Gaps Identified:** Isolated specific demographic sectors lacking access to primary care within a critical 5km buffer zone.
2. **Infrastructure Optimization:** Provided data-backed recommendations for locating temporary health outreach camps to maximize regional coverage.
3. **Data Integrity:** Corrected topological errors and overlapping polygon boundaries from legacy data, ensuring clean, usable geometric layers for regional leads.
