# 📊 Core Data Analytics & Geospatial Portfolio

Welcome! This repository serves as a centralized showcase of my work as a Data Analyst and Geospatial Analyst. I specialize in turning messy, disparate data—from web-scraped tables to complex geographic layers—into structured databases and interactive, insights-driven visualizations.

## 🛠️ Technical Toolkit

*   **Core Data Analytics:** SQL (Querying, Joins, Aggregations), Advanced Excel (Pivot Tables, VLOOKUP/XLOOKUP, Power Query), Python (Pandas, NumPy)
*   **Data Wrangling & Extraction:** Web Scraping (BeautifulSoup/Requests), API Data Extraction, Data Cleaning & Auditing, Formatting Anomalous Datasets
*   **Geospatial Intelligence:** QGIS, ArcMap, Vector/Raster Digitization, Spatial Analytics & Joining, Coordinate Reference Systems (CRS)
