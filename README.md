## 🏥 1. Public Health Outbreak Response Audit

### Business Problem

A public health vaccination campaign required operational performance analysis to identify coverage gaps, monitor vaccine hesitancy, and improve decision-making across multiple administrative levels.

### Solution

Developed an advanced monitoring and evaluation model in Microsoft Excel to audit vaccination performance, analyze campaign outcomes, and identify operational bottlenecks.

### Key Results

- Increased statewide vaccination coverage by **3.66%**
- Identified an **8.56% decline** in Doriga LGA
- Discovered a **39.6% increase** in father-led vaccine refusals
- Produced evidence-based recommendations for targeted interventions

### Tools Used

- Microsoft Excel
- XLOOKUP
- Data Cleaning
- Data Validation
- Reporting

🔗 **Project:** [Operational Campaign Audit](./project-marindi-immunization)
